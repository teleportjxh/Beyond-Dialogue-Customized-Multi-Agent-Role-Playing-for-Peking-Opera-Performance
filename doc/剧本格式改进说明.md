# 剧本格式改进说明

## 改进目标
解决原剧本格式混乱的问题，使动作、表情、念白、唱词清晰分离，对齐示例剧本格式。

## 主要改进点

### 1. 清晰的层次结构
**改进前：**
```
【第1场】
────────────────────────────────────────────────────────────
系统：
    【念】【场景开始】场景1：未命名

孙悟空：
    【念】【情】（机敏好奇，活泼灵动）
[〖急急风〗锣鼓声中...]
【念】（韵白）呔哈！...
【做】[一个"筋斗"...]
```

**改进后：**
```
### 第一场：初见

#### 场景设定
孙悟空因好奇心来到蜀地，偶然间遇到正在庭院中准备煮酒的诸葛亮。

#### 出场角色及顺序
1. 孙悟空
2. 诸葛亮

#### 表演流程

**【孙悟空出场】**
- (内心：机敏好奇，活泼灵动)
- [动作] 〖急急风〗锣鼓声中...
- [念白] 呔哈！...
- [动作] 一个"筋斗"...
```

### 2. 标签规范化
**改进前：**
- 混用【念】、【做】、【唱】标签
- 动作和台词混在一起
- 情绪描述位置不当

**改进后：**
- 统一使用 `[动作]`、`[念白]`、`[唱词]`、`[提示]` 标签
- 内容类型清晰分离
- 情绪描述独立标注为 `(内心：...)`

### 3. 内容分离
**改进前：**
```
【念】（韵白）[轻抚胡须，微微颔首] 悟空兄远道而来，不知有何指教？
【做】[手搭凉棚，作"瞭望"身段，转身以袖拭额]
```

**改进后：**
```
- [动作] 轻抚胡须，微微颔首
- [念白] 悟空兄远道而来，不知有何指教？
- [动作] 手搭凉棚，作"瞭望"身段，转身以袖拭额
```

### 4. 唱词格式优化
**改进前：**
```
(唱) **[西皮慢板]**
蜀地佳酿果然妙，
诸葛先生高论昭。
天下英雄何其多，
智勇双全才称豪。
```

**改进后：**
```
- [唱词] [西皮慢板]
  ```
  蜀地佳酿果然妙，
  诸葛先生高论昭。
  天下英雄何其多，
  智勇双全才称豪。
  ```
```

### 5. 移除系统信息
**改进前：**
- 包含"系统：【场景开始】场景1：未命名"等系统提示
- 场景名称显示为"未命名"

**改进后：**
- 移除所有系统提示信息
- 使用大纲中的场景名称（如"初见"、"论英雄"）

## 技术实现

### 修改的文件
- `src/script_generation/script_formatter.py` - 完全重写格式化逻辑

### 核心改进
1. **场景组织**：按场景号组织对话，跳过系统消息
2. **内容解析**：使用正则表达式分离锣鼓经、唱词、念白、动作
3. **标签统一**：将所有标签转换为统一的 Markdown 格式
4. **层次清晰**：使用 Markdown 标题（###、####）构建清晰层次

### 关键函数
- `_organize_by_scenes()` - 按场景组织对话
- `_parse_content()` - 解析对话内容，分离各类元素
- `_format_dialogue()` - 格式化单条对话
- `_format_single_scene()` - 格式化单个场景

## 使用方法

### 重新生成剧本
```bash
python regenerate_script.py
```

### 在主程序中使用
格式化器已集成到 `src/script_generation/main.py` 中，生成剧本时会自动使用新格式。

## 后续优化建议

1. **互动环节**：可以添加类似示例中的"互动"小节
2. **场景描述**：可以进一步丰富场景设定的描述
3. **角色行当**：在出场角色中标注行当（如"老生"、"武生"）
4. **锣鼓经细化**：根据不同场景使用不同的锣鼓经
