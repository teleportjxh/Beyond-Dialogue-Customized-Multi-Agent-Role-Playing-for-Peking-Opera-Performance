# 场景设定功能说明

## 功能概述

场景设定功能为京剧剧本生成系统添加了**布景描述**和**音效设计**，使生成的剧本更加完整和具有舞台表现力。

### 特点

- ✅ **可选功能**：默认关闭，不影响现有功能
- ✅ **向后兼容**：老代码无需修改即可正常运行
- ✅ **智能生成**：根据场景类型自动生成合适的布景和音效
- ✅ **失败保护**：即使场景设定生成失败，也不影响剧本生成

## 使用方法

### 1. 基本使用（不启用场景设定）

```python
from src.script_generation import ScriptGenerationSystem

system = ScriptGenerationSystem()

# 默认行为：不生成场景设定
result = system.generate_script(
    user_request="诸葛亮和孙悟空煮酒论英雄",
    max_scenes=3,
    max_rounds_per_scene=10
)
```

### 2. 启用场景设定

```python
from src.script_generation import ScriptGenerationSystem

system = ScriptGenerationSystem()

# 启用场景设定功能
result = system.generate_script(
    user_request="诸葛亮和孙悟空煮酒论英雄",
    max_scenes=3,
    max_rounds_per_scene=10,
    enable_scene_setting=True  # 启用场景设定
)

# 访问场景设定
scene_settings = result['scene_settings']
# 格式：{1: {scenery: "...", sound_effects: [...]}, 2: {...}, ...}
```

```markdown
**【布景】**
舞台中央布置一座简朴的茅庐，内有书架、竹席，墙上挂有几幅山水画。

**【音效】**
- 轻柔的古琴声作为背景音乐
- 偶有鸟鸣声点缀

#### 出场角色及顺序
1. 诸葛亮
2. 孙悟空

#### 表演流程
...
```

### 场景设定JSON文件

系统会额外生成一个 `{剧名}_场景设定.json` 文件：

```json
{
  "1": {
    "scenery": "舞台中央布置一座简朴的茅庐...",
    "sound_effects": [
      "轻柔的古琴声作为背景音乐",
      "偶有鸟鸣声点缀",
      "微风吹动竹叶的沙沙声"
    ]
  },
  "2": {
    "scenery": "...",
    "sound_effects": [...]
  }
}
```

## 场景类型与设定

系统会根据场景类型自动生成合适的布景和音效：

### 1. 开场场景
- **布景**：简洁大气，突出主要角色
- **音效**：庄重的锣鼓、引子音乐

### 2. 武打场景
- **布景**：开阔的战斗空间
- **音效**：激烈的锣鼓点、兵器碰撞声

### 3. 抒情场景
- **布景**：优美的自然景观
- **音效**：柔和的音乐、自然声音

### 4. 对话场景
- **布景**：适合对话的室内或室外环境
- **音效**：轻柔的背景音乐

### 5. 冲突场景
- **布景**：紧张的对峙空间
- **音效**：紧张的音乐、急促的锣鼓

### 6. 追赶场景
- **布景**：动态的追逐路径
- **音效**：快节奏的锣鼓、脚步声

## 注意事项

1. **默认关闭**：为保持向后兼容，场景设定功能默认关闭
2. **失败保护**：场景设定生成失败不会影响剧本生成
3. **API调用**：场景设定需要调用LLM API，会增加生成时间和成本
4. **可选输出**：只有启用场景设定时才会生成场景设定JSON文件

## 示例代码

### 完整示例

```python
from src.script_generation import ScriptGenerationSystem

# 初始化系统
system = ScriptGenerationSystem(
    character_dir="character",
    character_data_dir="character_data",
    vector_index_dir="vector_index",
    output_dir="generated_scripts"
)

# 生成带场景设定的剧本
result = system.generate_script(
    user_request="诸葛亮和孙悟空煮酒论英雄",
    max_scenes=2,
    max_rounds_per_scene=8,
    enable_scene_setting=True
)

# 查看结果
print(f"剧本标题：{result['outline']['title']}")
print(f"场景数量：{len(result['scene_settings'])}")

# 查看第一个场景的设定
scene_1 = result['scene_settings'][1]
print(f"\n第一场布景：{scene_1['scenery']}")
print(f"音效数量：{len(scene_1['sound_effects'])}")

# 输出文件
print("\n生成的文件：")
for file_type, file_path in result['output_files'].items():
    print(f"  {file_type}: {file_path}")
```
