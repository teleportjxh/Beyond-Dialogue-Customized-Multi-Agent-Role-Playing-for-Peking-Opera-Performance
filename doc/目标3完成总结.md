# 目标3完成总结：多Agent剧本生成系统

## 完成时间
2025年11月11日

## 目标概述
构建多Agent协作的京剧剧本生成系统，实现编剧、定妆师、演员、导演四个Agent的协同工作，生成具有京剧专业元素（身段动作、唱腔、锣鼓经等）的高质量剧本。

## 核心成果

### 1. 多Agent架构实现
成功实现了四个专业Agent的协作系统：

#### 编剧Agent (ScriptwriterAgent)
- **职责**：根据用户需求生成剧本大纲
- **输入**：用户需求、角色信息、RAG检索的相关场景
- **输出**：包含场景划分、情节发展的结构化大纲
- **特点**：利用RAG语义检索增强，确保生成内容符合角色特征

#### 定妆师Agent (CostumeAgent)
- **职责**：为每个场景的角色设计服装和造型
- **输入**：场景大纲、角色基础信息
- **输出**：详细的服装描述（颜色、款式、配饰）
- **特点**：考虑场景氛围和角色身份，提供视觉化指导

#### 演员Agent (ActorAgent) - **核心增强**
- **职责**：生成角色对话，智能添加京剧专业元素
- **输入**：场景上下文、其他角色对话、场景类型、是否首次登场
- **输出**：包含对话、情感、身段、唱腔、锣鼓经的完整表演内容
- **核心创新**：
  - **场景类型识别**：识别6种场景类型（开场、武打、抒情、叙事、对话、冲突）
  - **智能元素添加**：根据场景类型动态决定添加哪些京剧元素
  - **首次登场检测**：角色首次出现必须包含起霸、亮相、锣鼓经
  - **Few-shot学习**：使用真实剧本《金钱豹》片段作为示例
  - **动态场景调整**：根据对话轮次进度动态调整场景类型

#### 导演Agent (DirectorAgent)
- **职责**：评估剧本质量，提供改进建议
- **输入**：完整剧本内容
- **输出**：多维度评分（情节连贯性、角色塑造、对话质量、京剧元素、整体评分）
- **特点**：提供具体的改进建议，指导剧本优化

### 2. 京剧专业元素智能添加系统

#### 场景类型识别逻辑
```python
场景类型判断规则：
- 开场：第一场景 或 包含"开场/登场/上场"关键词
- 武打：包含"武打/战斗/打斗/交手"关键词
- 抒情：包含"抒情/感慨/回忆/思念"关键词
- 叙事：包含"叙述/讲述/回顾"关键词
- 冲突：包含"冲突/争执/对峙"关键词
- 对话：默认类型
```

#### 动态场景调整机制
```python
根据对话轮次进度动态调整：
- 首次登场：强制为"开场"类型
- 进度 < 30%：保持基础场景类型
- 进度 30-70%：可能转为"对话"或"叙事"
- 进度 > 70%：可能转为"抒情"（情感升华）
```

#### 京剧元素库
**身段动作**：
- 起霸：`[角色名拉起霸。]`
- 亮相：`[作威武"亮相"身段]`
- 筋斗：`[一个"筋斗"翻至台中]`
- 瞭望：`[手搭凉棚，作"瞭望"身段]`
- 拂袖：`[拂袖而去]`

**锣鼓经**：
- 急急风：`[〖急急风〗锣鼓声中]`
- 小锣：`[〖小锣声〗中]`
- 四击头：`[〖四击头〗]`

**唱腔板式**：
- 西皮：`(唱) **[西皮原板/慢板/快板]**`
- 二黄：`(唱) **[二黄原板/慢板]**`
- 吹腔：`(唱) **[吹腔]**`

**武打程式**：
- 起打：`[起打]`
- 对打：`[对打]`
- 叉阵：`[摆开一个"叉阵"]`

#### 场景类型与元素对应关系
| 场景类型 | 必须包含 | 应该包含 | 可选包含 |
|---------|---------|---------|---------|
| 开场 | 起霸/亮相、锣鼓经 | 上场动作 | 唱腔 |
| 武打 | 武打动作、急急风 | 对打程式 | 身段 |
| 抒情 | 唱腔（西皮/二黄） | 情感身段 | 锣鼓经 |
| 叙事 | 韵白 | 身段动作 | 唱腔 |
| 对话 | 对话内容 | 身段动作 | - |
| 冲突 | 对峙动作 | 锣鼓经 | 武打 |

## 核心创新点

### 1. 场景类型驱动的元素添加
不是简单地在每段对话中添加元素，而是根据场景类型智能判断：
- 开场场景：必须有起霸、亮相、锣鼓经
- 武打场景：必须有武打动作和急急风
- 抒情场景：优先使用唱腔
- 对话场景：适度添加身段动作

### 2. 动态场景调整机制
根据对话进度动态调整场景类型：
- 开场阶段：保持原始场景类型
- 中间阶段：可能转为对话或叙事
- 结尾阶段：可能转为抒情（情感升华）

### 3. Few-shot学习
使用真实剧本《金钱豹》的片段作为示例，让模型学习：
- 如何自然地插入身段动作
- 锣鼓经的使用时机
- 唱腔的格式和韵律

### 4. 首次登场特殊处理
角色首次出现时强制添加完整的上场程式：
```
[〖急急风〗锣鼓声中] → [迈着虎步上场] → [作威武"亮相"身段]
```
