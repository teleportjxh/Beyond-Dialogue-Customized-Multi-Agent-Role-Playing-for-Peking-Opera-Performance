# 变更日志

## [2.1.0] - 2025-11-24

### 🔧 重要修复

#### 迭代机制优化
- **修复问题**：修复了"视频越迭代越差"的问题
- **根本原因**：原有机制完全替换prompt导致场景和装扮信息丢失
- **解决方案**：采用"保留基础信息 + 应用改进建议"策略

#### 核心改进

1. **`src/extractor/simple_prompt_extractor.py`**
   - `_build_complete_prompt`函数添加`improvements`可选参数
   - `extract_prompts_simple`返回数据中添加`scene_desc`、`costume_desc`、`content_text`字段
   - 支持在保留场景和装扮的基础上应用改进建议

2. **`src/evaluator/director_agent.py`**
   - `generate_improved_prompt`方法重构，接收`scene_desc`和`costume_desc`参数
   - 调用`_build_complete_prompt`重建完整prompt，而非让大模型自由生成
   - `analyze_and_improve`方法添加场景和装扮参数传递

3. **`src/pipeline/intelligent_pipeline.py`**
   - `_generate_video_with_director_guidance`方法提取并保存基础信息
   - 确保场景和装扮在所有迭代中保持不变
   - 调用评估时正确传递这些关键信息

#### 效果对比

**修复前**：
```
迭代1: 完整prompt (场景+装扮+表演)
迭代2: 简化prompt (丢失场景和装扮) ← 质量下降
```

**修复后**：
```
迭代1: 完整prompt (场景+装扮+表演)
迭代2: 完整prompt (场景+装扮+表演) + 改进建议 ← 质量提升
```

## [2.0.0] - 2025-11-19

### 🎉 重大重构

完全重写了整个项目，实现了基于导演Agent的智能视频生成系统。

### ✨ 新增功能

#### 1. 导演Agent评估系统
- **多维度评估**：5个维度（装扮准确性、动作质量、对话表达、场景一致性、整体印象）
- **智能反馈**：分析视频问题并给出具体改进建议
- **迭代优化**：根据反馈自动改进prompt并重新生成
- **质量保证**：只有通过7分以上的视频才会被接受

#### 2. 智能Prompt提取系统
- **动态提取**：从generated_scripts文件夹自动读取三个JSON文件
- **多源融合**：整合对话历史、装扮设计、场景设定
- **精准控制**：生成10-15秒短视频，聚焦单个人物的一次对话/动作
- **对话驱动**：通过对话历史指导视频生成，避免无关内容

#### 3. 模块化架构
- **extractor模块**：负责从剧本数据提取prompt
- **evaluator模块**：导演Agent评估系统
- **generator模块**：Sora2视频生成器
- **pipeline模块**：协调整个生成流程

#### 4. 统一入口
- **main.py**：命令行接口，支持多种参数
- **灵活配置**：支持自定义脚本路径、输出路径、迭代次数等
- **单视频测试**：支持只生成指定turn的视频

#### 5. 完善的文档
- **README.md**：完整的项目文档
- **QUICKSTART.md**：5分钟快速开始指南
- **config.example.py**：配置文件示例
- **test_system.py**：系统测试脚本

### 🔄 核心改进

#### Prompt提取逻辑重写
**之前的问题**：
- 直接使用用户提供的prompt，没有从数据中提取
- 没有整合装扮、场景、对话等多源信息
- 生成的视频内容不可控

**现在的实现**：
```python
# 从三个JSON文件提取信息
对话历史.json → 动作/对话内容（核心）
装扮设计.json → 角色外观描述
场景设定.json → 环境描述（简洁版）

# 组合成完整prompt
prompt = f"""
{角色装扮描述}
{场景描述}
{动作/对话描述}

重点：10-15秒短视频，单个人物的一次对话/动作，不出现其他东西
"""
```

#### 导演Agent角色重新定义
**之前的错误理解**：
- 导演只是"流水线检测员"
- 简单评估打分，3次失败就放弃
- 没有提供改进指导

**现在的正确实现**：
- 导演是"质量监督者和指导者"
- 对视频进行多维度评估并给出具体改进建议
- 根据反馈改进prompt，重新生成
- 迭代直到导演满意（或达到最大迭代次数）

#### 迭代机制优化
**之前**：
```
生成 → 评估 → 失败3次 → 放弃
```

**现在**：
```
生成 → 评估 → 分析问题 → 改进prompt → 重新生成
↑                                              ↓
└──────────────── 循环直到通过 ────────────────┘
```
