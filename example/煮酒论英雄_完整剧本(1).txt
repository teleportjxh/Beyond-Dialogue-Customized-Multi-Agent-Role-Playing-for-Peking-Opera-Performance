## 标题：《煮酒论英雄》

### 第一场：茅庐相遇

#### 场景设定
舞台中央布置一座简朴的茅庐，内有书架、竹席，墙上挂有几幅山水画。灯光柔和，营造出傍晚的宁静氛围。窗外竹影摇曳，偶有鸟鸣声。

#### 出场角色及顺序
1. 诸葛亮（老生）
2. 孙悟空（化作凡人，武生）

#### 表演流程

**【诸葛亮出场】**
- [动作] 诸葛亮手持羽扇，缓步慢行，步入茅庐，整冠理髯，轻摇羽扇。
- [唱词] [西皮导板]
  ```
  我本是卧龙岗散淡的人，
  今日得闲在此书中沉。
  竹影摇曳伴书声，
  闲看风云变幻影。
  ```
- (暗自思忖：此地清幽，正是读书好时光)

**【孙悟空出场】**
- [动作] 孙悟空化作凡人，灵活跳跃，手搭凉棚，四下观望，步入茅庐。
- [唱词] [西皮流水]
  ```
  云游四海到此地，
  见此茅庐隐逸奇。
  不知何人居于此，
  试探一番见高低。
  ```
- (心中暗想：此人非凡，定有高才)

**【互动】**
- [动作] 孙悟空拱手行礼，诸葛亮从容不迫，回礼。
- [念白]
  孙悟空：这位先生，茅庐清雅，敢问高姓大名？
  诸葛亮：在下诸葛孔明，不知阁下何人？
- [动作] 孙悟空抓耳挠腮，笑道。
- [念白]
  孙悟空：我乃云游之士，今日偶至此地，特来拜访。

**【锣鼓经】**
- [提示] [四击头]

### 第二场：煮酒论道

#### 场景设定
茅庐外，月色朦胧，一炉火正旺，酒坛摆放在旁，酒香四溢。灯光转为柔和的蓝色，营造出夜晚的氛围。

#### 出场角色及顺序
1. 孙悟空
2. 诸葛亮

#### 表演流程

**【孙悟空先登场】**
- [动作] 孙悟空灵活跳跃，金箍棒耍于手中，随后坐下。
- [唱词] [二六板]
  ```
  月明风清夜正好，
  煮酒论道共逍遥。
  英雄何处见真章，
  悟空心中有奇招。
  ```

**【诸葛亮登场】**
- [动作] 诸葛亮缓步走出，手持羽扇，举杯对饮。
- [唱词] [二六板]
  ```
  英雄之道智勇全，
  非止勇力称英贤。
  吕布虽勇终无智，
  智谋之士可称先。
  ```

**【互动】**
- [动作] 二人对坐，举杯对饮。
- [念白]
  孙悟空：先生所言有理，然我以为勇力亦不可少。
  诸葛亮：勇力固然重要，然智谋方能治国安邦。
- [动作] 孙悟空点头，若有所思。

**【锣鼓经】**
- [提示] [慢长锤]

### 第三场：英雄辩论

#### 场景设定
月下，酒香四溢，桌上酒杯已空。灯光柔和，营造出深夜的氛围。

#### 出场角色及顺序
1. 诸葛亮
2. 孙悟空

#### 表演流程

**【二人同时登场】**
- [动作] 二人起身争论，指点江山。
- [唱词] [快板]
  ```
  诸葛亮：
  智勇双全方为上，
  治国安邦才称王。
  孙悟空：
  俺老孙大闹天宫威名扬，
  勇力无双无人挡。
  ```

**【互动】**
- [念白]
  诸葛亮：勇力虽强，然治世需德才兼备。
  孙悟空：先生所言极是，然我仍有疑虑。
- [动作] 二人对视，心生敬佩。

**【锣鼓经】**
- [提示] [快长锤]

### 第四场：互相折服

#### 场景设定
黎明将至，炉火渐熄，舞台灯光逐渐变亮，象征新的一天开始。

#### 出场角色及顺序
1. 诸葛亮
2. 孙悟空

#### 表演流程

**【二人同时登场】**
- [动作] 二人相互作揖，互赠礼物。
- [唱词] [散板]
  ```
  诸葛亮：
  悟空勇力世无双，
  愿赠锦囊助君行。
  孙悟空：
  孔明智谋天下扬，
  愿赠金箍表心诚。
  ```

**【互动】**
- [念白]
  诸葛亮：愿君勇力智谋兼备，行走天下。
  孙悟空：愿先生才识长存，治国安邦。
- [动作] 二人互相拱手，心生敬佩。

**【锣鼓经】**
- [提示] [四击头]

### 总结
通过《煮酒论英雄》这一选段，展现了诸葛亮与孙悟空之间的思想碰撞和互相钦佩，体现了智勇双全的英雄标准。
